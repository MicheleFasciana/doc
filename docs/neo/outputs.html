    NEO Outputs¶

    NEO output files are produced only if SILENT_FLAG = 0.

    All NEO runtime information is written to out.neo.run.

    Standard output files¶

    • out.neo.equil
    • out.neo.f
    • out.neo.grid
    • out.neo.phi
    • out.neo.theory
    • out.neo.theort_nclass
    • out.neo.transport
    • out.neo.transport_gv
    • out.neo.vel
    • out.neo.vel_fourier

    Experimental profiles output files¶

    Produced only if PROFILE_MODEL = 2.

    • out.neo.transport_exp <out.neo.transport_exp
    • out.neo.expnorm

    Rotation output files¶

    Produced only if ROTATION_MODEL = 2.

    • out.neo.rotation

    Subroutine output¶

    When neo is run in subroutine mode, the outputs are contained in a monolithic file named neo_interface. The NEO subroutine output parameters are here <out.neo.subroutine.
