Parameters for input.cgyro¶

Simple example

# Numerical Resolution
N_ENERGY=8
N_XI=16

N_THETA=24
N_RADIAL=4

DELTA_T=0.01

KY=0.3

# Geometry
EQUILIBRIUM_MODEL=2
RMIN=0.5
RMAJ=3.0
Q=2.0
S=1.0

# Profiles
N_SPECIES=2

Z_1=1
MASS_1=1.0
DLNNDR_1=1.0
DLNTDR_1=3.0

Z_2=-1
MASS_2=0.0002724486
