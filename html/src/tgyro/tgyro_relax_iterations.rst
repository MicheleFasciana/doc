TGYRO_RELAX_ITERATIONS
----------------------

**Definition**

Maximum number of Newton iterations. 

**Comments**

- Setting TGYRO_RELAX_ITERATIONS=0 is useful to generate initial fluxes and other profiles on the TGYRO grid.

**Internal name**

tgyro_relax_iterations

**Default**

0
